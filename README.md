# 微信小程序－电影推荐


## 数据接口
- 豆瓣api https://developers.douban.com/

## 说明

- 实现热映，待映，搜索，口碑等功能
- 实现图片集水平滚动
- 实现下拉加载
- 实现设置功能，可清除缓存、编辑个人资料、查看手机信息、更新地理位置
